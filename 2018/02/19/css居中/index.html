这个方法实现垂直居中的原理是，每行文字都存在一个行框，行框包括上边距和下边距以及文本高度，当行框和父元素的高度一样的时候，那么行框的上下边距会变大，但是文本高度不变，因为上下边距是相等的，所以实现了垂直居中的效果。

块级元素

高度未知

1.通过margin和position实现居中

效果如下:

原理是设置了宽度和高度之后，蓝色方块不会主动去填满红板，而margin:auto会根据红板的大小，自动调整，从而实现居中。

2.通过transform实现

原理是通过设置top,left，把绝对定位把蓝色方块的左上交定位在红色定位的中间，再设置transform把蓝色方块的中心偏移到红板的中心。

高度已知

原理是通过设置top,left，把绝对定位把蓝色方块的左上交定位在红色定位的中间，再设置margin把蓝色方块的中心偏移到红板的中心。

flex-box

上面总结了很多中实现水平居中和垂直居中的方法，他们都有自己适用的地方，但是随着css3的出现，flex提供了一种轻松实现水平和垂直居中的方法。

只需要三行代码就实现了水平垂直居中，确实是十分的方便，但是因为是css3带来的新特性，所以在兼容性方面还是有一些不足的，为了适配不同的浏览器常常需要加不同的前缀。
